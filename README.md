# Dragon Slayer

A short web-based text RPG made mainly using HTML and JavaScript that utilises the concept of randomness (RNG).

![Thumbnail](Images/Thumbnail.png)

**Demo**

https://nauticos.github.io/Dragon-Slayer/

**Features**

* D20 rolls (like in D&D) to determine player damage
* 4 types of enemies with different stats
* Completely RNG based (no skill required)
* Interactive menu with clickable buttons

</br>

Made as a submission for Hack Club Entropy.
